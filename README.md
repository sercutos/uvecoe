# openECOE-Desktop
# Descargar proyecto:
 `git clone <tu-repo> `

 `cd <tu-repo> `
## Instalación de dependencias
 `npm install ` → instala todas las dependencias

## Desarrollo
###  PASO 1: Abrir consola:
 `npm run dev `
### PASO 2: Abrir otra consola:
## Iniciar aplicación:
 `npm start ` → inicia la app de Electron
 
## Subir cambios al repositorio de Github
## Descargar los cambios del repositorio de Github a Local
Credenciales:
`git config --global credential.helper store `
Update;
`git pull `
